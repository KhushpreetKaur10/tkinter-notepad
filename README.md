# Tkinter Notepad 📝

**Tkinter Notepad** is a clean, lightweight text editor built using **Python 3** and **Tkinter**. This desktop application demonstrates GUI development, file handling, and event-driven programming skills — perfect for showcasing practical Python experience on your resume or portfolio.

---

## 🚀 Key Features

- **New, Open, Save Files** – Works with `.txt` and all file types  
- **Text Editing** – Cut, Copy, Paste functionality  
- **Scrollable Text Area** – Handles long documents efficiently  
- **Custom Window Icon** – Polished desktop look  
- **About Section** – Info popup with developer details  

---

## 💻 Skills Demonstrated

- Python 3.x programming  
- Tkinter GUI design (`Text`, `Menu`, `Scrollbar`, `MessageBox`)  
- File I/O (read, write, save-as)  
- Event-driven programming & command binding  
- Error handling & user-friendly notifications
